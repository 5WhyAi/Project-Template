SharePoint Site Creation Guide for 5WHY Digital Products

Step-by-Step Instructions

  1. Log in to Office 365 Admin Center using your admin credentials.
  2. Navigate to SharePoint Admin Center.
  3. Click on "Active sites" and then "Create".
  4. Select "Team site".
  5. Configure site details:
    • Site Name: Enter the name of the company (e.g., "5Why Strategies Knowledge Base").
    • Site Address: Auto-generated based on the site name.
    • Group Email Address: Auto-generated.
    • Site Description: Provide a brief description (e.g., "Knowledge base for 5Why Strategies").
    • Privacy Settings: Choose "Private - Only members can access this site".
  6. Add owners and members, then click "Finish".
  7. Repeat the steps for each company.
