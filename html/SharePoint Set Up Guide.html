SharePoint Setup Guide for Multiple Businesses

Step 1: Create a SharePoint Site for Each Business

  • Log in to Office 365 Admin Center: Go to admin.microsoft.com and log in with your admin credentials.
  • Navigate to SharePoint Admin Center: In the left sidebar, click on "Admin centers" and then "SharePoint".
  • Create a New SharePoint Site:
    • Click on "Active sites" and then "Create".
    • Select "Team site" or "Communication site" based on your needs.
    • Configure the site details:
      • Site Name: Enter the name of the company.
      • Site Address: Auto-generated based on the site name.
      • Group Email Address: Auto-generated.
      • Site Description: Provide a brief description.
      • Privacy Settings: Choose "Private - Only members can access this site".
    • Add owners and members, then click "Finish".

Step 2: Set Up Document Libraries and Folders

  • Access Each SharePoint Site: Navigate to each newly created SharePoint site.
  • Create Document Libraries:
    • Click on "New" and select "Document library".
    • Name the library (e.g., Internal Processes, Product Information, Customer Support).
  • Create Folders within Libraries:
    • Inside each document library, create folders to further organize documents.
    • Example Structure for Lucky Fig:
      • Document Library: Internal Processes
        • Folder: Supplier Information
        • Folder: Product Listings
      • Document Library: Product Information
        • Folder: Product Descriptions
        • Folder: Images
      • Document Library: Customer Support
        • Folder: FAQs
        • Folder: Support Tickets

Step 3: Integrate with Power Automate

  • Access Power Automate: Go to flow.microsoft.com.
  • Create Automated Flows:
    • Click on "Create" and choose "Automated flow".
    • Choose a trigger (e.g., "When a file is created in a folder").
  • Configure Actions:
    • Add actions to automate tasks such as moving files, sending notifications, or updating records.
    • Example: When a new product listing is added to the Lucky Fig document library, send an email notification to the marketing team.
  • Save and Test Flows: Save the flow and test it to ensure it works as expected.

Step 4: Sharing Products Across Brands

  • Use SharePoint Links:
    • For products that you want to share across brands (e.g., a brand board template), store the document in one SharePoint site and share the link with other sites.
    • Set appropriate permissions to allow access.
  • Power Automate for Cross-Site Sharing:
    • Create a flow in Power Automate to copy or move files between SharePoint sites when they are added or updated.
    • Example: When a new design asset is added to Lucky Fig Creations, copy it to the 5Why Global Strategies document library.

Step 5: Automating Dropshipping Tasks

  • Product Research and Supplier Management:
    • Use tools like Oberlo, Spocket, or AliExpress to find products and suppliers.
    • Integrate these tools with SharePoint using APIs or webhooks to automatically update supplier information and product listings.
  • Listing Optimization:
    • Use Power Automate to create flows that automatically update product listings with optimized descriptions, images, and tags.
    • Integrate with SEO tools to fetch and apply best practices for listing optimization.
  • Advertising Automation:
    • Use Power Automate to schedule and manage advertising campaigns.
    • Integrate with advertising platforms like Google Ads or Facebook Ads to automate ad creation, budgeting, and performance tracking.
