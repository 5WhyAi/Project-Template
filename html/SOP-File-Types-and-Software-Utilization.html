L&J Solutions Standard Operating Procedure (SOP)

File Types, Knowledge Base Management, and Software Utilization

This document outlines the standard file types, knowledge base management practices, and software utilization guidelines to be used across all operating companies under L&J Solutions for optimal compatibility and accessibility.

Standardized File Types

File Category Primary Format Secondary Format Purpose
Documents PDF DOCX, HTML PDF for final versions, DOCX for editable documents, HTML for GPT compatibility
Spreadsheets XLSX CSV XLSX for data manipulation, CSV for data transfer
Presentations PPTX PDF PPTX for editable presentations, PDF for finalized versions
Graphics PNG/JPEG SVG PNG or JPEG for images, SVG for scalable vector graphics
Video MP4 - MP4 for wide compatibility

Software Utilization for Knowledge Base Management

  • Microsoft Office: Use Word, Excel, and PowerPoint for document creation and editing. SharePoint integrates seamlessly with these tools for real-time collaboration.
  • Adobe: Use Adobe Acrobat for PDF management, Illustrator for vector graphics, and Photoshop for image editing.
  • Microsoft Teams: Integrate Teams with SharePoint for seamless communication and collaboration. Create channels for different projects and link to relevant SharePoint document libraries.
  • SharePoint: Utilize SharePoint's document libraries, metadata, and search capabilities for efficient document management.

Additional Recommended Software

  • Trello: For project management
  • Asana: For task tracking
  • Miro: For collaborative brainstorming sessions

Guidelines for File Management

  • Always save final versions of documents in PDF format for sharing and archiving.
  • Use version control for editable documents (e.g., DOCX, XLSX, PPTX).
  • Create HTML versions of all final documents using TextEdit for GPT compatibility.
  • Include HTML conversion instructions in all SOPs.
  • Compress large files before sharing or uploading to the knowledge base.
  • Use descriptive file names following the company naming convention.
  • Store all files in the designated knowledge base or document management system (SharePoint).
