Landing Page for the Project Template

Welcome to the project template. Your folder should be set up in the following manner:

  • docs/ - Documentation and guides
  • html/ - HTML files
  • css/ - Stylesheets
  • js/ - JavaScript files

Each bullet point should be its own folder inside the main project folder. Proper structuring is important, and the case of the words (e.g., lowercase) matters.

Anytime an update is done, especially to styles or something like that, make sure to update the respective files.
